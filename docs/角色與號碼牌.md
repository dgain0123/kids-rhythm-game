# 角色與號碼牌（js/characters.js）

畫面中間的可愛角色。**數量＝該關要打幾下**，每隻頭上有號碼牌（第幾隻）。

## 角色清單 characters/index.json
```json
{
  "characters": [
    { "id": "cat",  "name": "貓咪", "img": "cat.png" },
    { "id": "dog",  "name": "小狗", "emoji": "🐶" }
  ]
}
```
- `img`：放在 `characters/` 的圖片（建議透明背景 PNG、正方形）。
- `emoji`：用 emoji 當角色，不用放圖。
- 目前名單：貓咪、兔子（去背真圖）＋ 小狗/恐龍/小熊/老虎/貓頭鷹/章魚/烏龜（emoji）。

## 選角色
- 標題下方**角色選單**，點一下換角色；選擇存 `localStorage`（下次記得）。
- 也可用網址 `?char=<id>` 指定預設。

## 數量＝要打幾下
- `setCharCount(n)`：進關時由 `main.js` 以 `chart.maxHits` 呼叫。
- `renderCurrent()` 把選中的角色複製 n 隻並排，每隻包一層 `.char-item.char-<id>`，內含號碼牌 `.char-num`（1、2、3…）。
- 過關時角色換成 n 個 🎉 拉炮（數量對應）。

## 號碼牌
- 樣式：白底、粉紅框、**圓角正方形**、大數字（`.char-num`）。
- 版面：預設**直式**——號碼牌在上、角色在下（`.char-item` flex column），不重疊。

## 每個角色可個別調號碼位置
`.char-item.char-<id>` 這個 class 讓某角色能客製。目前**兔子**特別處理：
```css
.char-item.char-rabbit .char-media { max-height: 72%; }  /* 兔圖縮小，上方留白 */
.char-item.char-rabbit .char-num {
  position: absolute; top: 0; left: 35%; transform: translateX(-50%);
}
```
- 兔子耳朵很高、間隙窄，號碼牌放**兩耳正中間、正上方**。
- `left: 35%` 是**量測出來的兩耳間隙中心**（左耳中心 13%、右耳中心 57% → 中點 35%）。
- 驗證方法：用 Playwright(webkit=Safari 引擎) 抓 `.char-num`/`.char-media` 的實際渲染座標比對，或截圖檢視（見 [踩過的坑](踩過的坑.md) 的「用截圖驗證版面」）。

## 白色角色的可見度
白色角色(如兔子)在淺背景會看不清 → `img.char-media` 加了淡 `drop-shadow`。

## 新增角色（約 30 秒）
1. 圖片放進 `characters/`（透明 PNG、正方形）。
2. `index.json` 加一筆 `{ id, name, img }`（或用 emoji）。
3. 重整即可；若要離線也有，記得把圖加進 `sw.js` 的預抓清單並同步 -public。
