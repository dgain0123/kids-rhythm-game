# 踩過的坑（除錯筆記）

一路遇到的問題與最終解法，避免重踩。

## 麥克風
- **Mac mini 沒內建麥克風** → 系統輸入完全沒訊號，別誤判成權限/程式。先 `system_profiler SPAudioDataType` 看有沒有實體輸入(虛擬的 Teams/Zoom 不算)。要接外接麥克風。
- **開麥克風的初始化爆音**被算成一下 → 加 `warmupMs=600` 暖機期忽略。
- **快速打兩下只算一下** → `refractoryMs` 太長(400ms)把第二下吃掉，改 130ms，防餘音改靠遲滯武裝。
- **換關量表全 0** → `resume()` 沒喚醒被 suspend 的 AudioContext；串流死掉沒重開。已修，並加「重連麥克風」按鈕。
- **麥克風常當掉**：外接 3.5mm 易接觸不良；多個瀏覽器分頁搶麥克風也會卡；可 `sudo killall coreaudiod` 重啟音訊引擎。

## 音效（Safari 最麻煩）
- **過關沒歡呼、失敗有**：Safari 自動播放限制——麥克風觸發(非點擊)的 `<audio>` 被擋。解法：開始遊戲點擊時 `primeCheer` 靜音播一下解鎖元素。
- **Web Audio 在此機無聲**：改用 `<audio>` 元素播真人音效。
- **歡呼中間斷一下**：曾因(1)過關 close 麥克風→macOS 重設裝置閃斷(改 pause)、(2)彩帶 fillText 數百 emoji 卡主執行緒(改 sprite+延後)。
- **離線過關沒聲、失敗有**：大檔 `<audio>` 發範圍請求，離線快取只回完整檔→Safari 拒絕。解法：音效改 **blob 整包載入記憶體**播放，不再發網路/範圍請求；SW 另加範圍請求回 206 當保險。

## 版面 / UI
- **畫面捲不動、標題被切**：曾用垂直置中+overflow hidden；改 `flex-start`+可捲動+一律顯示捲軸。
- **快取一直給舊版**：改用禁快取 `serve.py`；瀏覽器仍頑固時清網站資料。
- **號碼牌蓋住角色**：改直式(號碼在上、角色在下)；兔子特例放兩耳正中上方。
- **用截圖驗證版面**：Playwright(webkit=Safari 引擎)開本機頁、截 `#face`/`#stage`，或 `getBoundingClientRect` 量實際座標——不靠肉眼猜。

## 部署 / 快取
- **改了線上卻沒變**：GitHub Pages 建置 + CDN 傳播要數分鐘。先 `curl` 線上檔確認含新內容、`gh api .../pages/builds/latest` 確認 build=built，再叫使用者重整。
- **Service Worker 更新**：常要載入一次才切換，重整 2 次或清網站資料。
- **serve.py 單執行緒卡死**：瀏覽器並發請求會卡（Playwright 會逾時），改 `ThreadingHTTPServer`。

## 工具
- macOS **沒有 `timeout` 指令**，別用。
- AI 會議室的 **GPT 常駐瀏覽器分頁會殘留舊對話**→答非所問；長 prompt 會打字逾時。用精簡逐字稿、掉線當棄權。
