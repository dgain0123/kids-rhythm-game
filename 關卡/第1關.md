# 第1關：打1下

> 這是「第1關」的唯一規格正解檔。此關要改規則，先改這裡，程式/資料以本檔為準。

## 目標玩法
- 畫面顯示：一個**黑色四分音符**（畫在五線譜第三間＝小鼓位置）
- 小朋友對麥克風**打1下鼓** → **過關** 🎉
- 打**第2下** → **失敗** 😢
- **不判拍子/時間點**（幾秒打都可以，只看打幾下）

## 過關 / 失敗判定
| 情況 | 結果 |
|---|---|
| 打 1 下，之後 1.5 秒內沒再打 | 過關 |
| 打到第 2 下（超過 maxHits） | 失敗 |

- 判定邏輯：打到 `maxHits`(=1) 下後開一個 1.5 秒視窗，視窗內沒再多打就過關；期間又打一下就失敗。
- 程式：`js/game.js`（狀態 ready → hitOnce → pass/fail）。

## 譜面資料
- 檔案：`charts/level1.json`
- 內容：
  ```json
  {
    "title": "第1關：打1下",
    "maxHits": 1,
    "hint": "看到音符，打1下鼓！只能打1下喔～",
    "notes": [ { "type": "quarter", "drum": "snare" } ]
  }
  ```
- `maxHits`：這關最多能打幾下（第1關＝1）
- `notes`：譜面音符（`type` 音符種類、`drum` 鼓件）

## 過關表現
- 中間角色換成 🎉 拉炮、灑滿螢幕彩帶、「太棒了」大字
- 播放 `sounds/cheer.*` 真人歡呼（沒有則用合成後備）
- 綠框閃爍；「再玩1次」按鈕出現在卡片內

## 相關檔案
- 譜：`charts/level1.json`
- 關卡邏輯：`js/game.js`
- 打鼓偵測：`js/audio.js`
- 串接/畫面：`js/main.js`、`js/render.js`

## 之後新增關卡怎麼做
1. 在 `charts/` 加 `levelN.json`（可用 `tools/mid2json.py` 從 Logic MIDI 產生）
2. 在 `關卡/` 加 `第N關.md`（複製本檔改內容）
3. 讓 `main.js` 能載入該關（目前寫死載 level1，多關時再加關卡選擇）
