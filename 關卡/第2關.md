# 第2關：打2下

> 這是「第2關」的唯一規格正解檔。此關要改規則，先改這裡，程式/資料以本檔為準。

## 目標玩法
- 畫面顯示：**兩個黑色四分音符**（都在五線譜第三間＝小鼓位置，橫向排開）
- 小朋友對麥克風**打2下鼓** → **過關** 🎉
- 打**第3下** → **失敗** 😢
- **不判拍子/時間點**（幾秒打都可以，只看打幾下）

## 過關 / 失敗判定
| 情況 | 結果 |
|---|---|
| 只打 1 下 | 顯示「再打 1 下～」繼續等（不失敗） |
| 打滿 2 下，之後 1.5 秒內沒再打 | 過關 |
| 打到第 3 下（超過 maxHits） | 失敗 |

- 判定邏輯（`js/game.js`）：每打一下 `hits++`；`hits < maxHits`→進 `progress`（鼓勵再打，不開視窗、不失敗）；`hits == maxHits`→開 1.5 秒視窗，沒再多打就過關；`hits > maxHits`→失敗。
