# 第7關：打7下

> 這是「第7關」的唯一規格正解檔。此關要改規則，先改這裡，程式/資料以本檔為準。

## 目標玩法
- 譜面：**六個相連的八分音符 + 一個四分音符**（共 7 個音符，都在五線譜第三間＝小鼓位置；相連的八分音符自動加符樑）
- 中間角色顯示 **7 隻**（角色數＝要打幾下）
- 小朋友對麥克風**打7下鼓** → **過關** 🎉
- 打**第8下** → **失敗** 😢
- **不判拍子/時間點**（八分/四分只是譜面外觀，實際只看打幾下；每個音符＝一下）

## 過關 / 失敗判定
| 情況 | 結果 |
|---|---|
| 打 1～6 下 | 顯示「再打 N 下～」繼續等（不失敗） |
| 打滿 7 下，之後 1.5 秒內沒再打 | 過關 |
| 打到第 8 下（超過 maxHits） | 失敗 |

## 譜面資料
- 檔案：`charts/level7.json`（`maxHits:7`；6 個 eighth + 1 個 quarter）
- 八分音符繪製與符樑在 `js/render.js` 的 `drawNotes`。
