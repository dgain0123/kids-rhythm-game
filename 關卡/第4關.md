# 第4關：打4下

> 這是「第4關」的唯一規格正解檔。此關要改規則，先改這裡，程式/資料以本檔為準。

## 目標玩法
- 畫面顯示：**四個黑色四分音符**（都在五線譜第三間＝小鼓位置，橫向排開）
- 中間角色顯示 **4 隻**（角色數＝要打幾下）
- 小朋友對麥克風**打4下鼓** → **過關** 🎉
- 打**第5下** → **失敗** 😢
- **不判拍子/時間點**（幾秒打都可以，只看打幾下）

## 過關 / 失敗判定
| 情況 | 結果 |
|---|---|
| 打 1～3 下 | 顯示「再打 N 下～」繼續等（不失敗） |
| 打滿 4 下，之後 1.5 秒內沒再打 | 過關 |
| 打到第 5 下（超過 maxHits） | 失敗 |

- 判定邏輯同其他關（`js/game.js`）：`hits<maxHits`→progress；`hits==maxHits`→開視窗；`hits>maxHits`→fail。

## 譜面資料
- 檔案：`charts/level4.json`（`maxHits:4`、四個 quarter/snare 音符）

## 導覽
- 第3關過關後按「下一關 →」進入；或用網址 `index.html?level=4`；或從關卡目錄點「第4關」。
- 關卡順序寫在 `js/main.js` 的 `LEVELS`。
